# 材质管理系统使用指南（简化版）

## 🎯 核心功能

MaterialConfig 系统**自动用 Toon Shader 替换所有材质**：

```
任何 Shader 的模型
    ↓
点击颜色预设（如"Red"）
    ↓
✅ 自动覆盖为 Toon Shader + 应用红色
```

---

## 🚀 快速使用

### 第一步：创建配置

```
Tools → Material → Create Material Config
```

✓ 自动创建 10 个颜色预设  
✓ 所有预设都配置为使用 Toon Shader

### 第二步：应用到模型

```
选中模型 → Window → Material Manager
    → 点击颜色预设（如"Red"）
```

✓ Shader 自动变为 Toon  
✓ 颜色自动变为红色

**完成！** 🎉

---

## 📋 10 个默认颜色预设

```
[0] White    → Toon + 白色
[1] Red      → Toon + 红色
[2] Green    → Toon + 绿色
[3] Blue     → Toon + 蓝色

[7] Cyan     → Toon + 青色

[9] Orange   → Toon + 橙色
```

## 💡 使用场景

### 场景 1：统一场景风格

需要：整个场景都用 Toon Shader

```
1. Create Material Config
2. 全选所有模型
3. 逐个点击对应的颜色预设
4. ✓ 完成！全部变成 Toon Shader
```

### 场景 2：角色着色

需要：给 10 个角色不同颜色，都用 Toon Shader

```
1. Create Material Config
2. 角色 1 → 点 "Red"    → Toon + 红色
3. 角色 2 → 点 "Green"  → Toon + 绿色
4. 角色 3 → 点 "Blue"   → Toon + 蓝色
...
10. 完成！都用 Toon Shader + 各自颜色
```

### 场景 3：批量更新

需要：把所有混乱的 Shader 统一为 Toon 风格

```
1. Create Material Config
2. 选中所有需要改的对象
3. 批量应用相同颜色
4. ✓ 所有 Shader 都变成 Toon
```

## ⚙️ 代码使用

```csharp
// 获取组件
MaterialApplier applier = myObject.GetComponent<MaterialApplier>();

// 应用某个预设（自动覆盖为 Toon Shader）
applier.SetAllMaterialsToConfig(1); // 应用预设 1 (Red)

// Console 会显示：
// ✓ 已应用材质 'Red' 到 N 个渲染器
// Shader 已覆盖: XXX → Toon Shader
```

## 🔄 如何修改颜色或更新预设？

### 修改某个预设的颜色

1. 选中 MaterialConfig.asset
2. 在 Inspector 中找到该预设
3. 修改 **Base Color**
4. 保存
5. 重新应用到模型

### 自定义添加新预设

MaterialConfig 支持无限数量的预设，直接在 Inspector 中修改 Materials 数组即可。
